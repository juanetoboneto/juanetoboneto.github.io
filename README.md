# Acerca de este Blog:
Este proyecto es un blog dedicado a públicar artículos relacionados con matemáticas financieras, estadística, ciencia de datos, y cálculo actuarial que me parezcan útiles o interesantes y con fuentes formales.

# Créditos:
Por cuestiones de tiempo utilice como base la plantilla de blog de [Juan Camilo Moreno Gómez](https://github.com/JuanCamilo21) todo el rédito debido a él. Me encarge de modificar elementos estéticos y de estructura de archivos para adecuarlo a las necesidades particulares de mi blog.

## Tecnologías:
La página esta creada apartir de HTML y CSS además de la librería de JavaScript MathJax para renderizar fórmulas matemáticas en formato LaTex. Conforme avance la página ire agregando más funcionalidades.
